# FASO TICKET 🎫

Plateforme de billetterie événementielle et de transport interurbain au Burkina Faso.

## 🔐 Sécurité

Ce projet utilise des mesures de sécurité de niveau NASA :
- ✅ Obfuscation complète du code client
- ✅ Row Level Security (RLS) sur toutes les tables
- ✅ Rate Limiting sur toutes les API
- ✅ Content Security Policy (CSP) stricte
- ✅ Pas de données sensibles dans le client
- ✅ Source maps désactivées en production
- ✅ Console logs supprimés en production

## 🚀 Installation

\`\`\`bash
npm install
npm run dev
\`\`\`

## 📦 Build Production

\`\`\`bash
npm run build:secure
\`\`\`
